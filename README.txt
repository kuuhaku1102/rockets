Akihabara Rockets WordPress Theme
=================================

This is a custom WordPress theme designed for Akihabara Rockets Co., Ltd.

## Installation
1. Upload the `rokets` folder to your WordPress `wp-content/themes/` directory.
2. Go to Appearance > Themes in your WordPress admin panel.
3. Activate the "Akihabara Rockets Theme".

## Features
- **Responsive Design**: Fully responsive layout for Desktop, Tablet, and Mobile.
- **Modern Animations**: Scroll-triggered reveal animations for sections.
- **Custom Business Sections**: Includes dedicated sections for About, Services (7 items), and Company Profile.
- **FontAwesome Integration**: Uses FontAwesome icons for service descriptions.
- **Google Fonts**: Uses 'Sora' and 'Inter' for modern typography.

## Configuration
- The menus can be managed via Appearance > Menus (Primary Menu).
- The logo is currently text/icon based in `header.php`. You can replace it with an image tag if needed.
- The contact form in `front-page.php` is a placeholder. Install a plugin like Contact Form 7 and replace the placeholder shortcode.

## Files
- `style.css`: Main stylesheet.
- `functions.php`: Theme setup and enqueuing.
- `header.php`: Header template.
- `footer.php`: Footer template.
- `front-page.php`: Homepage template (Static Front Page).
- `assets/js/main.js`: Javascript for animations and interactions.
